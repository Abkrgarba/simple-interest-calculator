# Simple Interest Calculator

A lightweight Bash script that calculates **Simple Interest** — created as part of an open-source Git & GitHub learning project.

## Description

This script computes simple interest using the standard formula:

```
Simple Interest = (Principal × Rate × Time) / 100
```

## Getting Started

### Prerequisites

- A Unix/Linux terminal or Git Bash on Windows
- Bash shell (version 3.2 or higher)

### Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/simple-interest-calculator.git
cd simple-interest-calculator
```

### Usage

Make the script executable, then run it:

```bash
chmod +x simple-interest.sh
./simple-interest.sh
```

You will be prompted to enter:
1. **Principal** – the initial amount of money
2. **Rate** – the annual interest rate (in %)
3. **Time** – the time period (in years)

The script outputs the calculated Simple Interest.

### Example

```
Enter the principal: 1000
Enter the rate of interest per annum: 5
Enter the time period in years: 3
Simple Interest = 150
```
